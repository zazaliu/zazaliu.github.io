概述

我们经常会在 python 文件中看到 if __name__ == '__main__': 这样的代码块，这段代码的含义基本可以概括为：

Make a script both importable and executable

意思就是：该文件既可以当作一个模块被其他文件导入也可以当作脚本文件直接执行。
